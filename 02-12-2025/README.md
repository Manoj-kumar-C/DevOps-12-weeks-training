# Manoj DevOps Tasks - Python Examples

This repository contains practical DevOps-oriented Python scripts, examples and CI/CD pipeline configurations.

## Structure
- venv_demo/requirements.txt — example dependencies
- utils/file_ops.py — reusable file utilities
- ssh_examples/ssh_client_oop.py — lightweight OOP SSH wrapper using Paramiko
- ssh_examples/file_transfer.py — SFTP upload/download examples
- ci_cd/github_actions/python-ci.yml — GitHub Actions example workflow
- ci_cd/jenkins/Jenkinsfile — Jenkins Declarative Pipeline
- ci_cd/pipeline_script.py — small pipeline driver script
- os_examples/os_info.py — OS introspection examples
- os_examples/run_commands.py — run shell commands via Python
- iac_examples/config_generator.py — simple IaC config generator
